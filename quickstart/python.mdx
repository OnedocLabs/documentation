## 2. Generate a PDF using HTML

The easiest way to generate a PDF using the `html` parameter

```python generate.py
import os
import Onedoc

Onedoc.api_key = os.environ["Onedoc_API_KEY"]

params = {
    "owner": "Auguste <onboarding@Onedoc.dev>",
    "html": "<html> <body> <h1>My First Heading</h1> <p>My first paragraph.</p></body> </html>"
}

pdf = Onedoc.pdf.generate(params)
with open('myfirstdoc.pdf', 'wb') as file:
    file.write(pdf)
```
